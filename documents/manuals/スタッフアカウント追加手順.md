# スタッフアカウント追加手順

**対象システム**: オンライン予約システム
**作業時間の目安**: 約5分

---

## 事前準備（作業前に確認）

- [ ] このPCに **Node.js** がインストール済みであること
- [ ] このPCに **fukumoto-reservation プロジェクトフォルダ** があること
  　→ `C:\Users\SM7B\Workspace\apps\fukumoto-reservation\`
- [ ] **Google Chrome**（または任意のブラウザ）が使えること
- [ ] Googleアカウント（Firebase管理者）でログインできること

---

## STEP 1　Firebaseでユーザーを追加する

1. ブラウザで以下のURLを開く
   **https://console.firebase.google.com/project/project-3040e21e-879f-4c66-a7d/authentication/users**

2. 右上の「**ユーザーを追加**」ボタンをクリック

3. 以下を入力して「**ユーザーを追加**」をクリック

   | 項目 | 内容 |
   |------|------|
   | メールアドレス | クライアントのメールアドレス |
   | パスワード | 任意のパスワード（8文字以上推奨） |

4. ユーザー一覧に追加されたことを確認する

> ⚠️ この時点ではまだ管理者権限がありません。STEP 2 に進んでください。

---

## STEP 2　管理者権限（adminクレーム）を付与する

1. **Windows Terminal**（またはコマンドプロンプト）を開く

2. 以下のコマンドを**1行ずつコピー＆ペースト**して実行する

```
cd C:\Users\SM7B\Workspace\apps\fukumoto-reservation\functions
```

```
node -e "const admin=require('firebase-admin');admin.initializeApp({projectId:'project-3040e21e-879f-4c66-a7d',credential:admin.credential.applicationDefault()});admin.auth().getUserByEmail('【メールアドレスをここに入力】').then(u=>admin.auth().setCustomUserClaims(u.uid,{admin:true})).then(()=>{console.log('完了！');process.exit(0);}).catch(e=>{console.error(e.message);process.exit(1);});"
```

> **【メールアドレスをここに入力】** の部分を STEP 1 で登録したメールアドレスに書き換えること
> 例：`'fukumoto@example.com'`

3. ターミナルに `完了！` と表示されれば成功

---

## STEP 3　動作確認

1. ブラウザのシークレットウィンドウ（または別ブラウザ）で以下を開く
   **https://kojinius.jp/apps/OnlineAppointSystem/login.html**

2. STEP 1 で登録したメールアドレス＋パスワードでログイン

3. **管理画面（ダッシュボード）が表示されれば完了** ✅

---

## うまくいかない場合

### 「完了！」が出ずエラーになる

```
Error: Could not load the default credentials.
```
→ 以下を実行してから STEP 2 をやり直す

```
gcloud auth application-default login
```

ブラウザが開くので Google アカウントでログインする。

---

### ログインできない（管理画面に入れない）

- パスワードを間違えていないか確認
- STEP 2 が正常に完了しているか確認（`完了！` が出ていたか）
- ブラウザのキャッシュをクリアしてから再度ログイン
